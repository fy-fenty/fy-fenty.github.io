开始之前

React 是一个模块化的UI框架，本引导以一个常见的信息管理系统为例子，结合webpack一起使用。

相关技术

  • nodejs: 方便于本地环境的开发和调试，整合下面的技术。
  • webpack: 类似于gulp的构建工具，但是webpack本身有模块化功能，所以与React契合度比较高。
  • JSX: React里面的VDOM(Virtual DOM)结构语法，比较原始JS更加简洁。
  • es2015: javascript的最新版本，语法更简洁，引入了很多新特性，如模块、箭头函数、变量定义。手册
  • babel: javascript转换工具，将ES6，JSX转换为浏览器能够识别的内容。

第一步: 搭建nodejs

下面是node的配置，

  }
}

第二步：搭建webpack

				webpack.config.js
module.exports = {
    entry: {
        index: "./app"
    },
    output: {
        path: "./build",
        publicPath: "/static/",
        filename: "[name].js"
    },
    resolve: {
        extensions: ["", ".js", ".jsx"]     引入本地模块时，省去后缀名
    },
    module: {
        loaders: [{
            test: /\.jsx?$/,                将所有的js或jsx后缀的文件交给babel处理
            loaders: [ 'babel' ],
            exclude: /node_modules/
        }]
    }
};

第三部：组织基本组件

				module/Row.jsx
import React from 'react'             ES6特性，引入整个react类库

const Item = React.createClass({      使用React模块的createClass函数创建一个组件，也可采用继承的方法（class Item extends React.Component)
    render() {
        return (
            <div>                     return 的内容必须是只能有一个根节点
                这是一条记录
            </div>
        )
    }
});

export default Item;
表格模块，用于显示数据列表的容器 
				module/Grid.jsx
import React from 'react'
import Row from './Row';

var Grid = React.createClass({
    render() {
        return (
            <div>
                <Row />
                <Row />
                <Row />
                <Row />
            </div>
        )
    }
});

export default Grid;
程序入口文件，将Grid渲染到页面中 
				app.jsx
import React from 'react';
import { render } from 'react-dom';                     引入 react-dom 类库中的 render 模块
import Grid from './module/Grid';

render(<Grid />, document.getElementById('example'));
